> ## 사용 기술 스택

### 협업 툴

- 미리캔버스
- 구글 시트

### 사용 언어

- FE(Front End)

  - JSP
  - CSS
  - Javascript
  - jQuery

- BE(Back-End)

  - Language: Java, Ajax
  - WAS(Web-Application-Server): Apache Tomcat 9.0
  - Database
    - RDBMS: MySQL 
  - Developer tool
    - Java: Spring
    - Server: aws

- 사용 디자인 패턴: **MVC 패턴**

### 사용한 오픈 API

- BootStrap
- 카카오 지도 API
